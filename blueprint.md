# Like Lion Leader Birthday Rolling Paper

## Purpose and Capabilities
This is a modern, interactive birthday rolling paper application dedicated to the Like Lion leader. It allows members to leave heartfelt messages, like existing ones, and toggle between light and dark modes.

## Features
- **Modern UI/UX:** Built with React and Tailwind CSS, featuring a sleek, responsive design.
- **Dark Mode Support:** Smooth transition between light and dark themes.
- **Message Board:** A grid-based layout for messages from members.
- **Interactive Message Creation:** Users can add their own messages via a stylish modal.
- **Featured Message:** A special highlighted message for extra impact.
- **Likes:** Interactive like button for messages.
- **Visual Effects:** Marquee animations, code-themed celebration card, and orange glow effects.

## Project Structure
- `src/App.tsx`: Main application component containing all logic and UI.
- `src/index.css`: Tailwind CSS directives and custom animations/styles.
- `tailwind.config.js`: Custom theme configuration for Like Lion branding.
- `index.html`: Base HTML structure with Google Fonts and Material Icons.
